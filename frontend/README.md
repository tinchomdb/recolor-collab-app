# Recolour Workflow Frontend

Vue 3 + TypeScript frontend for ticket queue, approved library, and partner overview workflows.

## Commands

- `npm run dev` starts the Vite development server.
- `npm run build` runs type build checks and creates a production build.
- `npm run test` runs unit tests once.
- `npm run test:coverage` runs tests with coverage enabled.
- `npm run lint` runs static type checks (`vue-tsc`).
- `npm run format` and `npm run format:check` format/check source files with Prettier.
- `npm run quality` runs lint + coverage checks together.
